<h1 align="center">Pokedex</h1>
<p align="center">
Este proyecto fue desarrollado como parte de la materia Programación II. Consiste en una aplicación web que implementa la conexión a una base de datos local utilizando PHP y MySQL, trabajando con el entorno de desarrollo XAMPP y administrando la base de datos mediante phpMyAdmin. El sistema utiliza Bootstrap para crear una interfaz moderna y responsiva, combinando HTML, CSS y JavaScript para mejorar la experiencia del usuario. Además, se implementa la interacción con la base de datos mediante PHP, utilizando métodos GET y POST, así como consultas que recuperan información mediante fetch_assoc(). El proyecto también incluye la implementación de un CRUD (Create, Read, Update, Delete) para gestionar registros dentro de la base de datos. Finalmente, todo el desarrollo se gestiona mediante un repositorio colaborativo en GitHub, permitiendo el control de versiones y el trabajo en equipo.
</p>
